Linux安装 VMware是一个虚拟PC的软件,可以在现有的操作系统上模拟出一个新的硬件环境,相当于模拟一台新的PC,以此实现一台机器上真正同时运行两个以上的操作系统. VMware特点: 不需要分区或重新开机就能在同一台PC上使用两种以上的操作系统 本机系统可以与虚拟机系统网络通信 可以设定并且随时修改虚拟机操作系统的硬件环境

使用VMware Workstation安装Linux虚拟机大概步骤 准备:创建一个新的虚拟机 … 修改一些硬件设置:设置网络连接模式 内存大小 … 也可以增加一些硬件配置 1.分区(创建自定义的分区结构) sd sda sdb sdc … 硬盘 hd hda hdb hdc … 空闲空间(20GB)来分区 管理员 root 根分区 / 必须划分的分区: 1)根分区 挂载点 / 文件系统类型 ext3 分区大小 5000MB 2)swap分区(虚拟内存) swap分区不需要挂载点(图形界面安装选择后,挂载点不可选) 文件系统类型 swap 分区大小 512MB (物理内存2倍) 划分以上两个就可以用了 3)暂时不把/user分区单独划分出来 可以把 /user 单独划分出来(也可以不分) 4)web分区 用来存放网站的 挂载点 /web 文件系统类型 /ext3 分区大小 5000MB 5)test分区 用来做练习 挂载点 /test 文件系统 /ext3 分区大小 1000MB 2.网络设置 Bridge NAT Host-Only linux主机 IP:192.168.0.10 MASK:255.255.255.0 windows主机找到虚拟网卡 VMware Network adapter VMnet1(host-only) IP:192.168.0.11 子掩网码:255.255.255.0 重启linux就可以通信了,用SSH远程连接,输入linux的IP地址:192.168.0.10,用户名:root,密码:***;

Vware Workstation 虚拟网卡(来源互联网) 小编刚接触系统集成的时候，因为身边资源有限，所以只能在 VMware 虚拟环境上做一些测试。现在最新 VMware Workstation V8 已经发布了，此次更新不仅在原版安装文件上进行精简，保持了原有的品质，而且完美支持 32 位和 64 位系统，在功能和易用性方面有了很大提升。 　　虚拟机安装完成之后，VMware 会在主机上虚拟出两块网卡来，在 windows2003 的网络属性中可以看到多了 VMware Network adapter VMnet1(host-only)和 VMware Network adapter VMnet8(NAT)，这两块网卡就是虚拟机与主机通讯的桥梁，我们可以想象为主机上的一块网卡，是与虚拟机相连的。 　　host-only：模式相当于网卡直连 　　bridged：模式相当于通过 hub 　　NAT：相当于通过路由。 　　VMware Workstation 里有以下几种网卡模式： 　　1. birdged：桥接模式 　　VMware Workstation 上的虚拟机就像是网络上的一台独立主机，可以访问网络上任何一台机器，有自己独立的 IP，甚至可以通过 gateway，route 访问其他网络上的机器。 　　2. NAT：地址转换模式 　　VMware Workstation 上的虚拟机可以与主机共享一个 IP 　　3. host-only：主机模式 　　VMware Workstation 上的只能与主机通讯(就相当于两台机器通过双绞线互联) 　　4. custom：自定义模式 　　小编试着使用了 host-only 和 bridged 两种方式，以下分别说明： 　　一、在 VMware Workstation 中将网络设置为 host-only 　　1. 在虚拟机 linux 系统上设置网卡的IP为 192.168.0.10 　　2. 在宿主机 windows2003 中设置 VMware Network adapter VMnet1(host-only)的 IP 为 192.168.0.11 　　3. 将 windows2003 和 linux 重启，便可以通信了 　　二、在 VMware Workstation 中将网络设置成 bridged 　　1. 在虚拟机 linux 系统上设置网卡的IP为 192.168.0.10 　　2. 在宿主机 windows2003 中设置物理网卡 IP 为192.168.0.11 　　3. 将 windows2003 和 linux 重启，便可以通信了 　　在 bridged 模式北街还测试了同一网段中的其它服务器也可以访问 VMware Workstation 的 linux 系统，同样 linux 也可以访问局域网的其它服务器。 　　看到这里，我相信各位可以把 VMware Workstation 的网络搞定了。 注：更多请关注电脑教程栏目，三联电脑办公群:189034526欢迎你的加入 来源： http://www.3lian.com/edu/2013/05-13/69611.html
